# Sunny Soaking Massage — Cute Edition

English-only, cute single-page site with emoji and a 10-stamp loyalty program ($20 off).

## Deploy on GitHub Pages
1. Create a public repo (e.g., `sunny-soaking-cute`).
2. Upload all files (keep `assets/`).
3. Settings → Pages → Source: `main` branch, root folder.
4. Visit `https://<username>.github.io/sunny-soaking-cute/`.
