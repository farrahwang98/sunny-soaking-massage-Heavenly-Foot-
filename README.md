# Sunny Soaking Massage — Clean Cute Edition

English-only, cute single-page site with emoji. **No photos are included or displayed**. Uses the price list and business hours you provided, but not embedded as images.

## Deploy on GitHub Pages
1. Create a public repo (e.g., `sunny-soaking-clean`).
2. Upload all files.
3. Settings → Pages → Source: `main` branch, root folder.
4. Visit `https://<username>.github.io/sunny-soaking-clean/`.
